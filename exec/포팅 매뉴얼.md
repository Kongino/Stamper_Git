

1. 프로젝트 개발 환경
    - OS : Windows 10
    - IDE : IntelliJ(2022.1.3), Visual Studio Code(1.69.0)
    - DB : MySQL (5.7.35)
    - Server : AWS EC2 Ubuntu (20.04 LTS), nginx(1.18.0)
    - 기술 스택
        1) Backend : Java, JPA, Spring Boot
        2) Frontend : React, Node.js, Teachable Machine, kakao map api, Redux
